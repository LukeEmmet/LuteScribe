Quick start tutorial 

This page provides a simple quick start guide for creating a new tabulature project in LuteScribe.

Create a new project

Start LuteScribe and create a new project using the File->New menu

This creates a new blank project containing a single stave.

At the end of the file is the end of content marker "e".

Edit the tabulature

We now can enter some tabulature into the grid that defines the stave content. The top row sets the TAB command, and the rows below determine each course. Editing the cells in the grid works in a similar way to editing a spreadsheet.

We can start the piece with a new bar line, so enter "b" into the top row of the first stave and press [Return] to exit the editing of that cell. the selection will advance to the next element.

Now enter "1" into the second element like this to create make a flag with one "tail"

When you press return it will display using the custom font for flags like this:

Now in the cells below that flag type "a" into each of the top rows, corresponding to the first and second course.

You can click on each cell to edit it, or use your cursor keys to move between cells.

Now complete the notes and flags as below. Use the numbers 0 to 5 in the top line to set the number of tails on each flag (0 = zero tails, 1 = 1 tail etc).

These are the characters as you type them:

b  1 2   1   b 0 0 b  e
  a  c  d  c      a  a     
  a      d  c    a  a     
        d             
  c      a      c  c

Setting the piece title

Now you can change the title of the piece - as follows:

Click on the "Headers" tab and change the title from "{Untitled}" to  "{Simple LuteScribe demo piece}" (without quotes) in the list of headers. Then click on the last header row to update the list of headers.

Save your file

Now you can save the file as LSML using the File->Save As...  menu

Save it to your desktop as a file called "demo_piece".

Print preview

Now you can preview the output, using the File->Print preview menu. This will show the typeset output of the current piece in the third Print view tab.

Within that view you can then save a PDF file or print the output to your preferred printer.

Congratulations you have now created your first piece of tabulature using LuteScribe!
