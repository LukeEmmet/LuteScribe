TAB commands

You can insert any valid TAB command into the editing region or header area.

N.B You don't need to insert x as a flag to indicate continue same flag as before. This results in a cleaner and simpler editing experience.

For a full set of Tab commands, please consult the following PDF files:

  • TAB Manual by Leonard Williams - provides an in-depth discussion of the various formatting options offered by TAB, with many examples
  • TAB Manual Appendix  - a full list of all commands supported by TAB, which is part of the TAB distribution.

Both of these files are available via the Help menu in LuteScribe.

TAB header commands

Each TAB file has a number of associated header commands which define the piece title and a number overall settings for the piece. Enter these into the headers section. Read the TAB file PDF documentation for more details

Some useful header commands are provided via the Headers menu which is shown when you are editing the headers, covering aspects such as:

  • Tabulature style
  • Page layout
  • Playback
  • Flag and note styles
  • Printing and layout

To set the piece title and author, enter a header as follows:

  • {Title of piece}
  • {Title of piece/Author}

Other headers can be found in the TAB documentation.

TAB stave commands

Within the stave editing region you can set the command or flag for any chord.

In historical tabulature, the top line of each chord indicates the flag associated with the current note or chord, and then below are shown the frets and courses of each chord. Similarly in LuteScribe, stave commands can be inserted into the stave lines directly by editing the relevant cell. So the main difference from standard TAB is that each row is rotated for ease of playing,

To set an element in the first column in plain text Tab, enter the corresponding command into the top row of the chord.

On the Insert menu, a menu of common stave commands is available for rapid selection (see image below). These are example Tab commands and are not a full list of those you can choose.  Each of these is flag style command which will be applied to the currently selected chord.

Some useful TAB commands are available from the Chord menu:

  • Blank chord and bar lines
  • Bars and breaks
  • Rhythm elements
  • Time signatures
  • Markings and fourishes

Other commands can be found in the TAB documentation
