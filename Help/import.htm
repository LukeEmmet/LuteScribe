Import notes

LSML

  • full support (native XML format for LuteScribe)

TAB

  • all tabulature commands are recognised
  • no support for dual tabulature and mensural/standard notation staves
  • multi-section files can be detected if the sections are separated by a comment in the TAB content: %===new section===

FT3 - Fronimo file format

  • most chordal and rhythmic information imported
  • some fingering or ornamentation information may be ignored in the current version

FT2 - Legacy Fronimo file format

  • not recommended - very limited support for reading FT2 files
  • most content will need to be edited after loading

JTZ, JTXML - Fandango file format

  • can read multi-section files
  • all content is converted to French tabulature. If requried, you can - use the -h and -O TAB header options to automatically convert the output to italian tabulature.
  • basic chord and rhythmic information imported
  • some fingering or ornamentation information may be ignored in the current version
