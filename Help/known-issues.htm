Known issues and bugs

Bugs

  • Support for Fronimo FT2 files is very limited. Workaround: ask the file's author to generate a TAB or FT3 file, which will open better.
  • Visual formatting - complex bar lines (e.g. bB etc) do not display fully in stave view.

Possible future enhancements

The following might be implemented in the future, depending on level of interest. These are shown in a rough order of importance. Feel free to submit source code patches for these.

  • Provide warning and fail to try to open any TAB file containing both tabulature and standard notation.
  • Improved FT3 parsing to extract fingering, ornaments on each note.
  • improved FT3 parsing to extract other bar information such as bar line variations, repeat marks, time signature.
  • Keyboard navigation between staves (e.g. via cursor keys).
  • Keyboard option or user option to go to lower course after edit, not next chord on same course.
  • MIDI export - via relevant TAB option.
  • More user control over the fonts and styles.
  • Preview a selection of content using popup window to show what the current elements will render as.
  • More compact elements within stave (or user option)

Rejected future features constraints

The following features are unlikely to be implemented.

  • No support for TAB files containing both tabulature and standard notation.
