Editing

LuteScribe's main window presents a tabular input editor for each stave.

Keyboard navigation

Use your cursor keys to move around each stave. Typing in each cell will set its contents.

Inserting new chords

You can use the menu to insert a chord element after the currently selected element.

Visual formatting

LuteScribe aims to offer an editing experience that reflects the underlying Tab commands used, yet at the same time is displayed in a way that is familiar to tabulature readers. In this way it is possible to play directly from the tabulature you are editing.

To achieve this it uses some visual formatting rules for the top row of the tabulature which shows the command character. The rationale is to present the tabulature in a familiar way and to de-emphasise the display of common characters that are not needed to be read when playing the tabulature (e.g. b, B etc). The lines of tabulature that indicate the courses and frets use a historic style font (e.g. a b c d e f g 0 1 2 3 4 5)

Note that the "x" command of TAB to indicate "Continue previous flag" is not needed in LuteScribe. This results in a cleaner display and less editing for each note.
