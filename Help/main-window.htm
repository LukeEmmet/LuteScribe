Main Window

Tab has one main window containing a tab strip and menus.

Menus
See menus for details
Title
Just below the menu the title of the current file is displayed. It is editable and reflects the title set in the headers
Staves tab
This provides the staves of tabulature that you can edit - see editing
Headers tab
Allows you to edit or add the headers for the current file. This can contain the title, and other global formatting options such as font etc. - see editing headers
Print view
Shows the preview of the current file. From this tab you can print the piece or save a PDF. See Printing and saving PDF for details.
Options tab
Allows you to set any LuteScribe options such as stave wrap length - see options
