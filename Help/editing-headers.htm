Editing headers

On the second entry on the main window tab strip you can edit the headers of the current file. These are standard TAB headers, so any valid TAB header command can be inserted here.

To create a new header, put some text into the last row of the headers to create a new one.

To delete an existing header - select the header and press [Delete] on your keyboard

To edit an existing header - click on the entry and start typing to set its content

Setting the piece title

To set the title of the current piece, simply create a header entry enclosed in curly braces - e.g. {My piece title}. This is the standard TAB command to set the title. This will be shown by LuteScribe just below the menu strip.
