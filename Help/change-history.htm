Change history

1.0.1

December 2018

  • Public release

0.3.3 Beta

December 2018

  • FT3 loading - better detection of fretted and open diapason strings
  • No longer need to type "x" to repeat the last flag. Simply leave the flag empty as you would see in tabulature
  • Rudimentary undo implemented - can now revert to previous snapshot taken whenever you run a menu command, paste or delete content
  • No longer requires two new lines between headers and content when loading TAB files
  • User option to set preferred default charset and flag style when loading TAB files without any explicitly set
  • Title and author now both set when loading FT3 or JTXML files

0.3.1 Beta

  • Better styling of first bar in each stave
  • New option to control whether the last stave is padded at the end with a flourish (otherwise can result in unattractive stretching of sparse content by TAB)
  • Default charset and flag style is thin robinson style flags and elements, unless explicitly set
  • PDF generation indicates piece number for multi-piece files
  • Improvements to JTXML parser
  • Added recent file list for convenience
  • Better handling of empty elements when generating TAB output
  • Minor change to LSML serialisation format (older LSML files still open), official LSML version 1.0
  • Can open multi-piece TAB files delimited with "%=== new section ==="
  • Section menu to navigate through pieces within same file (TAB, JTXML)
  • Added unit tests

0.2.4 Beta

Released 30 December 2017

  • Now works on Linux with Wine
    • Confirmed on Ubuntu
    • Recommended to use Winetricks to install Microsoft .NET 4.6 plus Gecko
  • New files can now be saved
  • Added experimental JTXML loader
  • Remembers the last UI tab chosen - e.g. editing or preview tab, making it better for using LuteScribe as a viewer

0.2.3 Beta

  • Installer creates application shortcut on desktop
  • Menus to view TAB help files (PDF) work better
  • Updated help
  • Can now save a copy of FT3 files as LSML or TAB
  • Each note now trims off any leading or trailing space to prevent offsetting to next course by TAB

0.2.2 Beta

Released 5 November 2017

  • Rebuilt to run correctly on Windows 10 64 bit
  • Improved performance for print preview

0.2.1 Beta

  • Bug fixes
  • Updated help
