LuteScribe Help

This is the user help guide for LuteScribe

What is LuteScribe?

LuteScribe is a Windows application for viewing and editing musical tabulature for lutes and related historical instruments. It offers an editing window to view and edit your tabulature in a format that is easy to understand. It also shows the fully rendered output. You can listen to the current piece using its audio playback.

Behind the scenes, LuteScribe works closely with the TAB program to create attractive typeset output. All the formatting commands of TAB can be used in your LuteScribe content.

LuteScribe can open tabulature written in the following formats: TAB, Fronimo (FT3, FT2), Fandango (JTZ, JTXML), and its own LSML format, based on XML.

You can preview your tabulature, or save as PDF or print it out - see preview, printing and saving as PDF.

Relationship to TAB

TAB is a mature tabulature typesetting program that uses a text based format to author the tabulature. The TAB input text format is related to the final typset and provides a wide range of powerful typesetting commands.

However the TAB text format is not particularly easy to read from as it is rotated and upside down compared to the final typeset content, and the visual appearance is not familiar to lute players. As a result most writers of tabulature tend to prefer a GUI editor that is dedicated to writing tabulature.

LuteScribe works in conjunction with TAB, whilst at the same time providing a familiar user interface so what you see during editing is close enough to final ouput format that you can play from - see screenshot below. See further details about how LuteScribe builds on the strengths of TAB and works around its shortcomings.
