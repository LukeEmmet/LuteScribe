Menus

Here we summarise the behaviour of LuteScribe's menu commands.

File

New
Creates a new blank file.
Open
Opens a file in one of the following formats: LSML, TAB, FT3, FT2. See import compatibility for details of any compatibility issues when loading TAB, FT3 or FT2 formats.
Save
Saves the current edited content as LSML (LuteScribe XML format) or TAB.
Save As
Saves the current content to a new location.
Export
Export as PDF - creates a PDF file within the same folder as the current tabulature file.
Note that some PDF viewers such as Adobe Acrobat lock the PDF file being viewed, in which case LuteScribe cannot overwrite the file. If it this occurs, simply close the file in Acrobat and then you can resave the PDF from LuteScribe
Print preview
Previews the current piece (shown in the third tab). See preview, printing and saving PDF for further details.
Print
Prints the current file to a chosen printer.
Recent files
Shows a list of the most recent files opened.
Exit
Exits LuteScribe

Edit

Undo
Reverts back to a recent snapshot of the content, created when running a menu command, paste, delete items.
Redo
Reverts the most recent undo.
Cut
Moves the content onto the clipboard for pasting.
Copy
Copies the currently selected chords onto the clipboard.
Paste
Paste the content of the clipboard before the currently selected chord.
Select all
Selects all the chords on the current stave.

Chord

Insert blank chord
Inserts an empty chord before or after the current selection
Insert bar line
Inserts a simple bar line before or after the current selection. An empty chord can also be converted to a bar line by setting its flag to "b" or "B"
Delete chord
Deletes the currently selected chord. You can also delete selected items using [DEL] on your keyboard.

Insert

<Various items>
Insert the chosen item after the current selection. Each item is a shortcut to a common TAB command. These and any other TAB commands can be inserted manually since you can also edit the current chord directly as needed.

Stave

Insert stave break
Inserts a new stave after the current selection. This new stave will contain any chords after the selection.
Join with next stave
Deletes the end of the current stave. As a result any chords on the following stave will be added to the end of the currently selected stave.
Insert new stave
Inserts an empty stave before or after the currently selected stave.
Delete stave
Deletes the currently selected stave. You cannot delete the last remaining stave in a piece - there must be at least one.

Tools

Reflow all
Reflows all the bars of the current file so to fit within the specified stave wrap value. The approach taken is that a whole number of bars are shown on each stave up to the stave wrap value.
Set grid flag styles
Updates all grid flags in the current file to use the preferred style. Both styles are supported by TAB, but the N# format is preferred by some as it shows the flag element first, so lines up better with the rest of the chord.
Remove comments from staves
Removes any chords that are comments, namely begin "%" on the flag line. This can result in simpler files. Comments in headers are not affected.
